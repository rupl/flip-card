3D <flip-card> element

Mouseover the cards to flip them over.

Y axis Heyo!
X axis Howdy!

Sample code

<flip-card axis="y">
  <front>Y axis</front>
  <back>Heyo!</back>
</flip-card>

Installation

Via bower:

bower install flip-card

Via git:

git clone https://github.com/rupl/flip-card.git
